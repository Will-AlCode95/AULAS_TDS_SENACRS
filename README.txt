# Visualg

O **Visualg** é um ambiente de desenvolvimento simples, voltado para a criação e execução de algoritmos em português estruturado. Com uma interface amigável e intuitiva, ele facilita o aprendizado de programação, especialmente para iniciantes. O Visualg permite criar e testar algoritmos de maneira prática, sendo ideal para estudantes e professores.

## Recursos

- **Interface em português**: Pensado para quem está começando a programar.
- **Simulação de algoritmos**: Permite visualizar a execução passo a passo.
- **Fácil de usar**: Interface intuitiva com comandos em português.

## Instalação

1. Acesse o site oficial do Visualg: [http://visualg3.com.br](http://visualg3.com.br)
2. Baixe o arquivo de instalação compatível com o seu sistema operacional (apenas Windows).
3. Execute o arquivo baixado e siga as instruções de instalação.

> **Nota:** O Visualg é compatível apenas com o sistema operacional Windows.

## Começando a Usar

1. Abra o Visualg após a instalação.
2. Clique em **Novo** para criar um novo arquivo de algoritmo.
3. Comece a escrever seu algoritmo em português estruturado.
4. Para executar, clique em **Executar** e veja o passo a passo da execução.

## Exemplo de Algoritmo

Abaixo está um exemplo simples de um algoritmo em Visualg que calcula a soma de dois números:

```algoritmo
algoritmo "SomaSimples"
var
   num1, num2, soma: real
inicio
   escreva("Digite o primeiro número: ")
   leia(num1)
   escreva("Digite o segundo número: ")
   leia(num2)
   soma := num1 + num2
   escreval("A soma é: ", soma)
fimalgoritmo
