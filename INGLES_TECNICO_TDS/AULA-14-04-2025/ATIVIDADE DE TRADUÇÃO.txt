O QUE É PYTHON?

PYTHON É USADO PARA, INTERPRETAÇÃO, ORIENTADO Á OBJETOS, LINGUAGEM DE PROGRAMAÇÃO ALTO NÍVEL COM SEMÂNTICAS DINÂMICAS, USADO PARA VÁRIOS PROPÓSITOS NA PROGRAMAÇÃO.

PYTHON UMA SERPENTE GRANDE , O NOME DA LIMGUAGEM DE PROGRAMAÇÃO PYTHON SURGIU DA ANTIGA BBC SÉRIE DE TELEVISÃO DE COMÉDIA CHAMADA MONTY PYTHON O CIRCULO VOADOR.

NESSE SUCESSO, O TIME MONTY PYTHON VIAJOU POR TODO MUNDO, INCLUINDO O HOLLYWOOD BOWL.

DESDE MONTY PYTHON É CONSIDERADO UM DE 2 NUTRIENTES FUNDAMENTAIS DA PROGRAMAÇÃO(O OUTRO É PIZZA), O CREADOR DO NOME PYTHON DIZ QUE É UMA HONRA AO SHOW DE TV.
